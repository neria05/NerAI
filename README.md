## ✨ תכונות

- 🌍 **תרגום הודעות נכנסות** — כפתור תרגום ליד כל הודעה, עם תרגום מיידי לעברית (או כל שפה אחרת)
- ⌨️ **תרגום לפני שליחה** — כתוב בעברית, תרגם לשפת הנמען ושלח, כולל אימות בתרגום חוזר
- 📚 **תרגום מרוכז** — תרגום כל ההודעות בשיחה בלחיצה אחת
- 🤖 **ניתוח שיחה עם AI** — ניתוח אווירה, נושאים, עמדות והמלצת תגובה מוכנה לשליחה
- 💬 **צ'אט מהיר** — פתיחת שיחה עם מספר טלפון בלי לשמור איש קשר
- 📤 **ייצוא שיחה** — שמירת השיחה כקובץ טקסט

## 🛠️ שירותים נתמכים

| שירות | תרגום | ניתוח AI | דורש מפתח API |
|--------|:-----:|:--------:|:-------------:|
| Google Translate | ✅ | — | לא (חינם) |
| DeepSeek | ✅ | ✅ | כן |
| OpenAI (וכל ממשק תואם) | ✅ | ✅ | כן |

אם שירות בתשלום לא מוגדר — התוסף עובר אוטומטית לתרגום Google החינמי.

## 📦 התקנה

1. הורד או שכפל את הריפו
2. פתח את עמוד ניהול התוספים בכרום: `chrome://extensions`
3. הפעל **מצב מפתח** (Developer mode)
4. לחץ **טען תוסף שלא נארז** (Load unpacked) ובחר את תיקיית הפרויקט
5. פתח את [WhatsApp Web](https://web.whatsapp.com) — זהו!

## ⚙️ הגדרה

1. היכנס לשיחה כלשהי ב־WhatsApp Web
2. לחץ על כפתור ⚙️ שנוסף לכותרת השיחה
3. בחר שירות תרגום ושפת יעד (ברירת מחדל: עברית)
4. לניתוח AI: הפעל את המתג והזן מפתח API של DeepSeek או OpenAI

מפתחות ה־API נשמרים בדפדפן שלך בלבד (chrome.storage) ולא נשלחים לשום שרת מלבד ספק ה־AI שבחרת.

## 📁 מבנה הפרויקט

| קובץ | תפקיד |
|------|-------|
| `content.js` | הליבה — כפתורי תרגום, ניתוח AI, מודאל הגדרות |
| `input-translate.js` | תרגום תיבת ההקלדה לפני שליחה |
| `api-services.js` | חיבור לשירותי התרגום וה־AI |
| `background.js` | Service worker — קריאות רשת (עקיפת CORS) |
| `quick-chat.js` | צ'אט מהיר למספר לא שמור |
| `fix-translation-service.js` | תיקוף הגדרות ו־fallback אוטומטי |
| `popup.html/js` | חלון הסטטוס של התוסף |
| `PROJECT.md` | מסמך ניהול הפרויקט — החלטות ומשימות |

## ⚠️ הערה חשובה

תוסף זה אינו קשור ל־WhatsApp או ל־Meta. השימוש בו הוא באחריות המשתמש בלבד, בהתאם לתנאי השימוש של WhatsApp.
